Website Screenshots in MacOS Menu Bar

Created by Wynter Jones

Take screenshots of any website with Desktop, Mobile or Thumbnails all in beautiful high rise and save instantly to any folder on your desktop.

Use the keyboard shortcute COMMAND+SHIFT+V when you have a URL copied in your clipboard to auto-paste into WebSnap.

Version: 1.0.0
OS: MacOS
Size: 59.1MB
Requires Google Chrome Browser to be Installed
